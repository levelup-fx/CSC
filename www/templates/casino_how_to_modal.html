Como utilizar a Grelha

Abaixo irá encontrar os tópicos de ajuda disponíveis. Para saír pressione o botão "Voltar".

Explicação geral

Na grelha do Casino, as colunas correspondem aos jogadores, e cada linha corresponde a um jogo.

A soma dos pontos de todos os jogadores em cada jogo tem de ser igual ou superior a 11.

Ganha o jogador que atingir 21 pontos (ou outro limite de pontos).

Colocar os pontos

Pressione a célula da grelha do jogador / jogo para o qual gostaria de inserir os pontos.

Utilizando o teclado do seu dispositivo, insira os pontos que esse jogador fez.

Alterar n.º de jogadores

Podemos jogar Casino com 2, 3 ou 4 jogadores ou equipas.

Para alterar a quantidade de jogadores, pressione o botão com o ícone da chave inglesa no canto superior esquerdo.

Na janela que surgir, escolha o número de jogadores e pressione "OK".

Alterar o limite de pontos

Por defeito, um jogador ganha quando atingir 21 pontos.

No entanto é possível alterar este limite - Pressione o botão com o ícone da chave inglesa no canto superior esquerdo.

Na janela que surgir, abaixo do número de jogadores, pressione a caixa de texto com o limite de pontos de uma partida.

Altere este valor para qualquer valor desejado utilizando o seu teclado.

Alterar nomes dos jogadores

Para alterar o nome de um jogador, pressione-o escreva o novo nome desejado utilizando o teclado do seu dispositivo.

Próximo jogo

Pressione o botão "Seguinte", no canto inferior esquerdo.

Irá surgir uma nova linha, que corresponde a um novo jogo, e poderá inserir novamente os pontos para cada jogador.

Números à direita

Pode ter reparado que, à direita de cada jogo existe um número. Este serve para mostrar a soma dos pontos de todos os jogadores para um determinado jogo.

A soma dos pontos de todos os jogadores de um jogo deve ser igual ou superior a 11. Se for inferior a este valor, aparecerá a vermelho. Caso contrário, aparecerá a verde.

Colocar as vitórias

As vitórias são atribuídas automaticamente pela aplicação. Ao pressionar o botão "Seguinte", se um dos jogadores chegar ao limite de pontos, este fica com mais uma vitória.

Se houverem dois jogadores com o limite de pontos, é considerado um empate - ganha quem tiver mais pontos no próximo jogo.

Se três ou mais jogadores tiverem o limite de pontos, a aplicação perguntará quem foi o vencedor.

Reiniciar tudo

Pressione o botão "Recomeçar", que está no canto inferior direito.

Ao fazê-lo, irá surgir uma janela de confirmação - Caso aceitar recomeçar, todos os pontos (e também as vitórias!) são colocados a zero, e começamos a partir do primeiro jogo.

Pontuação do Casino

Existe uma lista de pontuação do Casino, útil para consultar caso não se lembre ou não saiba como calcular os pontos.

Pressione o botão com o ponto de interrogação, no canto superior direito.

Escolha a opção "Pontuação do Casino" para ver como se calculam os pontos neste jogo.

Casino para não fazer pontos

Existe uma variante do Casino que consiste em não fazer as cartas, ou evitar fazer o maior número de Cartas possível. Aqui vence quem tiver o menor número de pontos, regra muito semelhante ao jogo das Copas.

Para jogar esta variante recomendamos então usar a grelha do jogo "Copas", e alterar a pontuação limite para 21 pontos ou outro valor à escolha.
