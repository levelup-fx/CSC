Como utilizar a Grelha

Abaixo irá encontrar os tópicos de ajuda disponíveis. Para saír pressione o botão "Voltar".

Explicação geral

Na grelha do King, as colunas correspondem aos jogadores, e cada linha corresponde a um jogo.

Os pontos de cada jogador / jogo são apresentados nas respetivas células.

À direita, é apresentado o total dos pontos. Para inserir a pontuação, coloque na linha do jogo respetivo.

Colocar os pontos

Os pontos são calculados automaticamente em função das Vazas, Copas, Damas, ou outros elementos inseridos. O utilizador tem apenas de inserir o número de vazas, Copas, Damas, Homens (ou outro tipo de elementos) recolhidos.

Pressione a linha do jogo respetivo e irá surgir a janela ilustrada acima.

Utilize os botões acima ou abaixo para aumentar ou diminuir o número de Vazas, respetivamente. Quando clicar em OK, voltará para a grelha principal onde serão mostrados os pontos de cada jogador.

Alterar nomes dos jogadores

Para alterar o nome de um jogador, pressione-o escreva o novo nome desejado utilizando o teclado do seu dispositivo.

Números à direita

Pode ter reparado que, à direita de cada jogo existe um número. Este serve para mostrar a soma dos pontos de todos os jogadores para um determinado jogo.

A soma dos pontos de todos os jogadores de um jogo deve ser igual a um determinado valor (que depende do tipo de jogo). Se for inferior a este valor, aparecerá a vermelho. Caso contrário, aparecerá a verde.

Os totais deverão ser:

- 260 pontos (Não fazer Vazas);
- 260 pontos (Não fazer Copas);
- 200 pontos (Não fazer Damas);
- 240 pontos (Não fazer Homens);
- 160 pontos (Não fazer Rei de Copas);
- 180 pontos (Não fazer as duas últimas vazas);
+ 325 pontos (Festas).

Reiniciar tudo

Pressione o botão "Recomeçar", que está abaixo da grelha.

Ao fazê-lo, irá surgir uma janela de confirmação - Caso aceitar recomeçar, todos os pontos são colocados a zero.

Pontuação do King

Existe uma lista de pontuação do King, útil para consultar caso não se lembre ou não saiba como calcular os pontos.

Pressione o botão com o ponto de interrogação, no canto superior direito.

Escolha a opção "Pontuação do King" para ver como se calculam os pontos neste jogo.
