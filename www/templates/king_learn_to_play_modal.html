Como jogar ao King

O King é um jogo de cartas que deve ser jogado com um único baralho e 4 jogadores. É composto por uma série de jogos com objetivos diferentes. Em cada jogo distribui-se o baralho completo pelos jogadores. No King, os naipes obrigam naipes - Isto é, se um jogador começa a jogar com uma carta do naipe Paus, os restantes jogadores também são obrigados a jogar cartas do naipe Paus (exceto se não tiverem Paus - aí podem "baldar-se" jogando a carta que quiserem).

No início, escolhe-se o jogador que joga primeiro. No jogo seguinte, o jogador ao lado joga primeiro. E assim sucessivamente.

Abaixo está a lista de jogos do King, com o respetivo objetivo.

No primeiro jogo, o objetivo é não fazer vazas. No segundo jogo, o objetivo é não fazer copas. A primeira jogada de copas apenas pode ocorrer quando um jogador só tem cartas de Copas. No entanto, os jogadores podem "baldar-se" de copas. No terceiro jogo, o objetivo é não fazer Damas. O jogo acaba quando as quatro Damas tiverem sido recolhidas. No quarto jogo, o objetivo é não fazer Homens - Isto é, Reis e Condes. O jogo acaba quando os oito homens tiverem sido recolhidos. No quinto jogo, o objetivo é não fazer o Rei de Copas. O jogo acaba quando essa carta tiver sido atribuída a um jogador. No sexto jogo, o objetivo é não fazer as duas últimas vazas. Os jogos a partir do sétimo são chamadas as "Festas". Quando a Festa é de um jogador (que será o "Festeiro"), este pode escolher entre jogar a Positivos (em que o objetivo é fazer vazas) ou a Negativos / Nulos (em que o objetivo é não fazer vazas).

Antes de se começar a Festa, os restantes jogadores oferecem vazas (para Positivos ou para Negativos) se quiserem. Cada jogador faz uma oferta de forma sequencial - Uma vaza, duas vazas, três... No entanto, cada oferta tem de ser superior à anterior (alternativamente, um jogador pode "Passar", não oferecendo nada).

No caso de um jogador oferecer 1 vaza para Negativos, o jogador seguinte, caso queira fazer uma oferta, deve oferecer 4 vazas para Positivos. Se um jogador oferecer 2 vazas para Negativos, o jogador só pode oferecer (se quiser) 6 vazas para Positivos.

As ofertas vão continuando até só haver uma oferta na mesa (sendo que os outros jogadores passaram). Aí o "Festeiro" pode aceitar ou não. Se aceitar, o jogador que ofereceu as vazas escolhe o trunfo (mas só se for para Positivos), e diz quem é o primeiro a jogar (se é ele ou o jogador seguinte). Se não aceitar, o "Festeiro" é que escolhe (se vai para Positivos ou Negativos), escolhe o trunfo (se for para Positivos) e escolhe quem joga primeiro (se é ele ou o jogador seguinte).

Se um jogador oferecer 3 vazas para Negativos ou 7 vazas para Positivos, o jogador da Festa é obrigado a aceitar, e acabam as ofertas.

No final, o jogador que fez a oferta, irá dar as suas vazas ao jogador da Festa, ou, no caso das Festas para Negativos, retirar as vazas que ofereceu ao jogador da Festa.

Cada jogador tem direito a uma festa, existindo então um total de quatro festas. Após a última festa, o jogo acaba e ganha quem tiver a pontuação mais alta.
